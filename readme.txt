**WP BFI** lets you manage featured images in a more efficient way, by adding intuitive tools to your WordPress interface.

== Description ==

Tired of setting up featured images one by one from the edit screen of your posts/pages? How much time have you lost repeating this redundant task lately?

**WP BFI** allows you to manage your featured images directly from any posts/pages list screen, hence cutting this time significantly.

You will never see setting featured image for your posts/pages as a drudgery again!

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins` directory, or install the plugin through the *Plugins* screen.
1. Activate the plugin through the *Plugins* screen.
1. Use the *Media* -> *WP BFI - Settings* screen to configure the plugin.
1. Use the posts/pages screens to manage your featured images easily.

== Frequently Asked Questions ==

= What does this plugin actually do? =

**WP BFI** makes it easier to manage your featured images by adding simple tools to the WordPress interface, hence the time saving.

= How does it work? =

New intuitive tools will be added to the posts/pages list screen, allowing you to manage featured images individually and promptly.

= What if I have custom post types? =

**WP BFI** checks for custom post types with the featured image option activated. If it is, **WP BFI** will be available.
